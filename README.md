Mathematical Analysis of Stock Market Volatility and Portfolio Vectors

Project Objective: To apply Linear Algebra, Calculus, and Statistics to financial data to understand risk and return.

Problem Formulation: We treat a collection of stocks as a multidimensional vector space. By calculating the rate of change (Calculus) and the correlation (Statistics) between these vectors, we aim to find a mathematically "optimal" balance that minimizes risk (variance).

Methodology: We follow the scientific method: Data Acquisition -> Mathematical Modeling -> Implementation -> Validation.
